SynthLib is an open source synth librarian that can be used to send, receive, and store
patches for external synthesizers via a MIDI interface. Currently the only supported
synths are the Alesis Andromeda A6 and the Lexicon PCM91 digital reverb.

The project is in working alpha stage.
